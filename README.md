Репозиторий для запуска admin части системы chat-boost

## Для уставновки frontend
Перейти в папку
```bash
cd frontend
```

Установить пакеты

```bash
yarn
```

Запуск в dev режиме

```bash
yarn dev
```


## Для уставновки backend


Перейти в папку
```bash
cd backend
```

Установить пакеты

```bash
yarn
```

Запуск в dev режиме

```bash
yarn prod
```
